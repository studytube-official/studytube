# StudyTube TikTok Demo

TikTok広告やCapCut編集用の縦長デモページです。

- Public URL: `https://studytube-official.github.io/studytube/ads/tiktok/`
- 尺: 約18秒ループ
- 推奨録画: 9:16
- テキストは最小限にして、説明文はCapCutで後から追加する前提です。
- 実在するYouTubeサムネイルやチャンネル名は使わず、すべて架空のサンプル表示にしています。

## 固定シーン

スクショや編集素材に使う場合は、次のクエリで場面を固定できます。

- `?phase=subjects` 教科選択
- `?phase=units` 単元選択
- `?phase=videos` 動画検索と保存
- `?phase=memo` 動画メモ
- `?phase=list` マイリストと並べ替え
- `?phase=sync` ログイン同期と履歴
